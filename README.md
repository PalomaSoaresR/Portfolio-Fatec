### 1° semestre | 2/2023
#### Porygon

Empresa parceira: [FATEC São José dos Campos - Prof. Jessen Vidal ](https://fatecsjc-prd.azurewebsites.net/)



Aplicação backend com interface de linha de comando (CLI).
O Sistema de Informação foi desenvolvido para uma instuição de ensino (PBLTeX) especializada na aplicação de cursos com o método PBL (Problem Based Learning), que viza ciclos de entregas, com scores parciais para ao final ser computado o FEE (Fator de Ensino Evolutivo).


A solução contempla o cadastro de alunos e professores, calendário acadêmico com datas de início e témino de semestre, criação de turmas e grupos, atribuição de notas (score) e feedbacks para cada aluno e exportação de relatório por turma.
